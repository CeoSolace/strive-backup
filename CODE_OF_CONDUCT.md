# Code of Conduct

## Our Pledge

We are committed to providing a respectful and harassment-free experience for
everyone participating in this project.

## Expected Behavior

- Be respectful and constructive
- Accept feedback gracefully
- Focus on what is best for the project
- Use welcoming and inclusive language

## Unacceptable Behavior

- Harassment or discrimination
- Trolling or personal attacks
- Publishing private information without consent
- Deliberate attempts to undermine project security

## Enforcement

Project maintainers reserve the right to remove, edit, or reject contributions
that violate this code of conduct.

## Scope

This applies to all project spaces, including:
- GitHub issues
- Pull requests
- Discussions
- Community channels
