# Security Policy

## Supported Versions

Only the latest released version of this project is supported with security
updates.

## Reporting a Vulnerability

If you discover a security vulnerability, **do not open a public issue**.

Please report it privately using one of the following methods:

- Direct message to the maintainer
- Private repository security advisory (if enabled)
- Secure contact channel provided by the project

When reporting, please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any relevant logs or proof-of-concept code

## Scope

This policy applies to:
- Core bot logic
- Security modules
- Dashboard components
- Telemetry or enforcement-related code

Third-party dependencies should be reported to their respective maintainers.

## Disclosure Policy

We follow a **responsible disclosure** process.  
Public disclosure should only occur **after** a fix or mitigation is available.

Thank you for helping keep this project secure.
