# Privacy Policy

## Overview

This project values transparency and user trust.  
This document explains how data may be handled by the software.

## Data Collection

The software **may now or in the future** collect limited technical information,
including but not limited to:

- Application or bot identifiers
- Version information
- Runtime environment details
- Abuse or misuse indicators
- License enforcement signals

## What Is NOT Collected

The software is **not intended** to collect:
- Message content
- Personal user data
- Private Discord conversations
- Authentication tokens
- Credentials of any kind

## Purpose of Collection

Any collected information is used solely for:
- Security monitoring
- Abuse prevention
- License enforcement
- Integrity verification
- Diagnosing large-scale malicious usage

## Data Handling

- Data collection, if present, is designed to be minimal
- Data is not sold or shared with third parties
- Retention is limited to what is necessary for security purposes

## User Responsibility

By using, modifying, or distributing the software, you acknowledge that
security- or license-related telemetry **may be present**.

If local laws require additional disclosures, it is the responsibility of the
operator to comply with them.

## Changes

This policy may be updated as the project evolves.
